# Run AI Model Locally

## Requirements:

1. Python 3.x
2. Libraries: json, requests
3. ollama

## Ollama installation:
Linux 'curl -fsSL https://ollama.com/install.sh | sh'
Window 'https://ollama.com/download/OllamaSetup.exe'
Mac 'https://ollama.com/download/Ollama-darwin.zip'

## Run ollama locally:
'ollama run <model>'
